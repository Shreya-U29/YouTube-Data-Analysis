# 📊 YouTube Data Analysis – Python Project

## 📌 Project Overview

This project analyzes **YouTube video data** to uncover patterns in **viewer engagement, sentiment, categories, and emoji usage** in comments. The goal is to understand what type of content performs well on YouTube and how audiences react to different videos.

Using **Python and data analysis libraries**, the project explores:

* Viewer engagement trends
* Sentiment of YouTube comments
* Emoji usage patterns
* Category-wise video popularity
* Relationship between virality and engagement

---

## 📊 Data Source

The dataset contains information about **YouTube videos and their comments**, including:

### Video Data

* `video_id`
* `title`
* `category_id`
* `views`
* `likes`
* `dislikes`
* `comment_count`
* `publish_time`

### Comment Data

* `comment_text`
* `likes`
* `replies`

The dataset was processed and cleaned for analysis using Python.

---

## 🛠 Tools & Libraries Used

* **Python**
* **Pandas** – Data cleaning and manipulation
* **NumPy** – Numerical operations
* **Matplotlib** – Data visualization
* **Seaborn** – Statistical visualization
* **Plotly** – Interactive visualizations
* **NLTK** – Sentiment analysis of comments
* **Emoji Library** – Emoji extraction and analysis

---

## 🔍 Analysis Performed

### 1️⃣ Sentiment Analysis of YouTube Comments

Using **NLTK's SentimentIntensityAnalyzer**, comments were classified into:

* Positive
* Neutral
* Negative

This helped identify how viewers emotionally react to different videos.

---

### 2️⃣ Emoji Analysis

Emoji usage in YouTube comments was extracted and analyzed to determine:

* Most frequently used emojis
* Emotional tone of discussions
* Correlation between emojis and viewer sentiment

Emoji usage helps capture **emotional reactions beyond text**.

---

### 3️⃣ Category Dominance on YouTube

Analyzed which **content categories dominate YouTube** by examining:

* Total number of videos
* Total views
* Engagement metrics

This helps identify the **most popular content niches**.

---

### 4️⃣ Viral Videos vs Engagement

Examined whether **viral videos actually generate higher engagement** by comparing:

* Views
* Likes
* Comment count

This analysis helps determine whether high views translate into **active audience interaction**.

---

### 5️⃣ Engagement Metrics Analysis

Calculated and compared:

* Likes-to-views ratio
* Comment-to-view ratio
* Overall engagement patterns

This reveals how viewers interact with different types of videos.

---

## 📈 Key Insights

* Positive sentiment dominates most YouTube comment sections.
* Certain content categories consistently attract higher engagement.
* Viral videos often generate high views but **not always proportional engagement**.
* Emojis are frequently used to express reactions, especially in highly engaging videos.
